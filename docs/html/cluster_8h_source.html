82 /* Message types.
83  *
84  * Note that the PING, PONG and MEET messages are actually the same exact
85  * kind of packet. PONG is the reply to ping, in the exact format as a PING,
86  * while MEET is a special PING that forces the receiver to add the sender
87  * as a node (if it is not already in the list). */
